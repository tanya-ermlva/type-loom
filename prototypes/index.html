Type Loom — prototypes

Sandbox for kinetic typography experiments. Each is its own self-contained app.

Compress
Ellipse force fields squeeze and push letters in a grid. Cascade clamping with smart drop, per-word color blobs, animation export.

Field Dither
Force fields decide letter presence, not position. Floyd–Steinberg / Atkinson / Sierra / Bayer threshold each cell. Halftone typography.

Gooey Poster

Typographic compositions where words flow with adjustable density, envelope curves, and edge stillness. Predecessor of Compress.

Pulse (Atom)
Two-line typographic composition pulsing between layout states. Per-line backgrounds breathe with the text. The atomic unit used by Stack and beyond.

Stack
Vertical infinite scroll of 4 pulse-atoms (blue, pink, mint, coral). Each atom's horizontal pulse phase is locked to the scroll cycle.

Bloom (Atom)
A circle that interpolates between rest and active states — dot stays put while a stroked outline grows on top to engulf it. Atomic unit for the bloom-stack composition.

Bloom Stack — DFD
56 bloom atoms placed at the DFD logo grid. Move the cursor: each atom's growth value comes from its distance to the cursor, eased through a falloff curve. Atom config shared with /bloom.
